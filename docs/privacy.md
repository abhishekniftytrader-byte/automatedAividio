# Privacy Policy — abhi-automation

_Last updated: July 18, 2026_

abhi-automation is a personal, single-user automation tool that publishes
the operator's own videos to the operator's own social media accounts.

1. **No user data collected.** The tool has no public users, no sign-up,
   and no interface for third parties. It does not collect, store, or
   process any personal data from anyone other than its operator.
2. **Platform tokens.** The tool stores API access tokens for the
   operator's own accounts, on the operator's own private server, used
   solely to publish and to read aggregate statistics of the operator's
   own content.
3. **No sharing.** No data is sold, shared, or transferred to any third
   party.
4. **No cookies or tracking.** The tool operates no website and uses no
   cookies, analytics, or tracking technologies.
5. **Contact.** Privacy questions: withabhishekgupta@gmail.com
